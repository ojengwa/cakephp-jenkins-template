Using the Job Template.

  1. Check out the cakephp2-template project from Git:

    $ sudo su jenkins
$ cd $JENKINS_HOME/jobs
$ git clone git://github.com/ojengwa/cakephp2-template.git cakephp2-template

  2. Reload Jenkins' configuration, for instance using the Jenkins CLI:

    $ java -jar jenkins-cli.jar -s http://localhost:8080 reload-configuration

  3. Click on "New Job or New Item".

  4. Enter a "Job name".
  5. Select "Copy existing job" and enter "cakephp2-template" into the "Copy from" field.
  6. Click "OK".
  7. Uncheck the "Disable Build" option.
  8. Fill in your "Source Code Management" information.
  9. Configure a "Build Trigger".
  10. Click "Save".
